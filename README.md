# CHAT EM TEMPO REAL

## Tecnologias usadas

- Nodejs
- Express
- Socket.io
- git

## Intalação 

### Linux


- Clone o repositório

`git clone https://github.com/jadson179/Chat-socketio.git`

- Entre na raiz do projeto

`cd Chat-socketio`

- Instale as dependências 

`npm install -y`

- Acesse http://0.0.0.0:3000
